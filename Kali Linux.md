# Kali Linux

---
Kali Linux is an exciting and powerful Linux distribution designed specifically for cybersecurity. 
It is developed, maintained, and funded by offensive security.

### **Ethical Hacking and Penetration Testing:**
Tools like **Metasploit and Nmap** are pre-installed, and they are easily capable of simulating cyber-attacks to find vulnerabilities in networks and systems. This also helps in defending our systems.

### **Wireless Network Auditing:**
Tools like **Aircrack-ng** that allow you to test the security of wi-fi networks, and will help in checking our network and learn wireless security practices.

### **Digital Forensics:**
Tools like **Autopsy and Foremost** are used for cybercrimes investigation.
